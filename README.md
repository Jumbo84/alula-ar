# عدسة العلا — تطبيق الواقع المعزز (PWA)

تطبيق ويب متكامل قابل للتثبيت على الجوال، يعمل بكاميرا وموقع وبوصلة حقيقية، ويشتغل دون إنترنت في وضع «جولة العلا».

---

## كيف تنشره على GitHub Pages وتحصل على رابط دائم (بدون برمجة)

اتبع الخطوات بالترتيب. الرابط الناتج دائم ومجاني ويعمل بـ HTTPS (وهذا ما يشغّل الكاميرا و GPS).

### 1) أنشئ حساباً ومستودعاً
1. ادخل **github.com** وأنشئ حساباً مجانياً (إن لم يكن لديك).
2. اضغط **+** أعلى يمين الصفحة ثم **New repository**.
3. في خانة **Repository name** اكتب: `alula-ar`
4. اختر **Public**، ثم اضغط **Create repository**.

### 2) ارفع ملفات التطبيق
1. في صفحة المستودع، اضغط **Add file** ثم **Upload files**.
2. اسحب **كل** هذه الملفات دفعةً واحدة وأفلِتها:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
   - `icon-maskable-512.png`
   - `apple-touch-icon.png`
3. اضغط **Commit changes** (الزر الأخضر بالأسفل).

### 3) فعّل GitHub Pages
1. من أعلى المستودع، افتح **Settings**.
2. من القائمة الجانبية، اضغط **Pages**.
3. تحت **Source**، اختر **Deploy from a branch**.
4. تحت **Branch**، اختر **main** و **/(root)**، ثم اضغط **Save**.
5. انتظر دقيقة، وحدّث الصفحة. سيظهر الرابط أعلى الصفحة بهذا الشكل:
   `https://اسم-حسابك.github.io/alula-ar/`

هذا هو رابطك الدائم. شاركه مع الجميع.

---

## كيف يجرّبه أي شخص

1. يفتح الرابط من متصفح جواله.
2. **للتثبيت كأيقونة** (اختياري): يضغط زر «تثبيت التطبيق» في الشاشة الأولى.
   - على آيفون: زر المشاركة في سفاري ثم «إضافة إلى الشاشة الرئيسية».
3. يضغط **«ابدأ التجربة»** ويمنح أذونات الكاميرا والموقع والبوصلة.
4. **داخل العلا:** يختار «تجربة حيّة (موقعي)» ويوجّه الكاميرا حوله.
   **خارج العلا:** يختار «جولة العلا» ليرى معالم العلا عبر كاميرته الحقيقية.

---

## للعرض على الإدارة

زر **«تشغيل العرض التلقائي»** في الشاشة الأولى يشغّل عرضاً يعمل وحده دون كاميرا أو إنترنت — مناسب لشاشة الاجتماع. لمس الشاشة ينقلك للتحكم اليدوي.

---

## ملاحظات

- البيانات الحيّة من OpenStreetMap (مجانية ومفتوحة). تحتاج إنترنت في الوضع الحيّ.
- وضع «جولة العلا» يعمل دون إنترنت بعد أول فتح (بفضل خاصية العمل دون اتصال).
- لأعلى دقة في تثبيت المعالم على الصخور والمباني، تُبنى لاحقاً نسخة أصلية (ARKit / ARCore).
